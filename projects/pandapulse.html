Overview

PandaPulse is a full-featured job search and hiring platform built in Django for CS 2340 (Objects and Design) at Georgia Tech. The platform functions similarly to LinkedIn - connecting job seekers with employers through structured profiles, job listings, and an application pipeline. It serves three distinct user roles: Job Seekers, Recruiters, and Administrators.

The project placed 3rd out of all submissions across both class sections, earning our team a final exam exemption. Beyond the 15 required user stories, our team designed and implemented over 20 original features that pushed the platform well beyond the baseline requirement.

Required Features (15 User Stories)

  • Job Seeker profile creation with headline, skills, education, work experience, and links
  • Job search with filters for title, skills, location, salary range, remote/on-site, and visa sponsorship
  • One-click job application with a personalized tailored note
  • Application status tracking: Applied, Review, Interview, Offer, Closed
  • Privacy controls on seeker profiles
  • Skill-based job recommendations for seekers
  • Interactive map view of job postings
  • Distance-based map filtering from current location
  • Preferred commute radius setting
  • Recruiter job posting and editing
  • Candidate search by skills, location, and projects
  • Kanban-style applicant pipeline for recruiters
  • In-platform messaging between recruiters and candidates
  • Email candidates through the platform
  • Saved candidate searches with new-match notifications
  • Candidate recommendations for recruiter job postings
  • Office location pinned on map for each job posting
  • Applicant location clusters on recruiter map view
  • Admin user and role management
  • Admin job post moderation and removal
  • Admin CSV data export for reporting

Original Features We Added

Panda Paul - AI Chatbot Mascot
An integrated panda mascot chatbot that lets logged-in users navigate the site, update profile info, apply to jobs, and message people through natural conversation.
Pulses - Short-Form Video Feed
Any logged-in user can post short videos to promote themselves or their company. Users can like pulses and posters can delete their own content.
Offer Comparison Tool
Applicants can compare multiple job offers side-by-side with custom-weighted categories including salary, location, visa sponsorship, and benefits.
Interview Calendar System
Applicants schedule interviews from time slots set by the employer, fully integrated into the application workflow on both sides.
One-Click Follow-Up
Applicants can send a polite follow-up message after a set number of days using pre-built professional templates.
Skill Endorsements
Employers can endorse applicant skills with one click after conducting interviews, adding social proof to candidate profiles.
Response-Time SLA Badge
Each employer displays a "responds in approximately N days" badge on their job listings so applicants know what to expect.
Resume Auto-Parser
Uploaded resumes are automatically parsed into profile fields with skill normalization for better job matching accuracy.
Confetti on Application Submit
Celebrates job application submissions with confetti and fanfare to make the experience feel rewarding.
Application Viewed Indicator
Applicants can see when an employer has opened and viewed their application in real time.
Private Notes
Both applicants and employers can attach private notes to applications that are only visible to themselves.
Rejection Feedback Templates
Structured rejection messages with optional personalized notes so candidates always receive meaningful feedback.
Applicant Activity Feed
A real-time feed showing applicants exactly what is happening with each of their applications as status changes occur.
Company Culture Pages
Company profiles featuring culture descriptions, perks, and an office map gallery so candidates can evaluate fit before applying.
Dark Mode
Full site dark/light mode toggle that persists per user across sessions.
Account Recovery Flow
Forgot username and forgot password buttons that send recovery links to the email address associated with the account.
Interview Feedback Workflow
Employers can leave structured internal feedback and decision rationale for each candidate after interviews.
Archived Rejections
Employers can archive rejected applicants to revisit them later if new positions open up, preserving the candidate pool.

Technical Approach

The platform was built entirely in Django, leveraging its ORM, class-based views, and template system throughout. The project emphasized object-oriented design principles - the core focus of CS 2340 - including clean model separation, reusable components, and a well-structured codebase that could scale with additional features.

The map features use an interactive mapping library to pin job locations and cluster applicants geographically. The Pulses video feed, Panda Paul chatbot, and offer comparison tool were all designed and built entirely by our team without external templates or starter code for those components.
