Overview

SACC Hoops March Madness is a custom-built platform designed to bring a professional, "ESPN-style" experience to the St. Alphonsa Syro-Malabar Forane Catholic Church's annual basketball tournament. In the past, tracking scores and player stats was a messy process involving paper sheets and fragmented group chats. I built this site to serve as a central home for the community, where every player and fan can see live updates, deep-dive into player statistics, and even participate in tournament predictions right from their phones.

The project covers the entire journey of the tournament. It starts with a streamlined registration and payment flow for players and moves into a live draft system for team captains. Once the games begin, the site transforms into a real-time leaderboard.

Key Features

Live Score Ticker
A scrolling score ticker runs across the top of every page showing live and recent game scores, updating in real time via Supabase Realtime.
Playoff Bracket
A full bracket view with auto-advancing results and live game status - updates automatically as scores are entered by the admin.
Team Pages
Each team has a dedicated page with full rosters, per-player stats, team colors, and record. Captains can manage their roster through the platform.
Fantasy Points System (FPTS)
A custom formula (PTS + REB x1.2 + AST x1.5 + STL x3 + BLK x3 - TO) ranks all players across the tournament, giving fans a way to follow individual performance.
Player Stat Leaderboards
Sortable leaderboard tables for FPTS, points, rebounds, assists, and steals. Every column is clickable to re-sort live.
Player Profile Pages
Individual profile pages per player featuring radar charts and stat bars built with Recharts, showing their performance breakdown across all categories.
Power Rankings
Weekly power rankings with movement arrows (up, down, same) showing team momentum across the tournament.
Pick'em Predictions
Fans can make predictions on upcoming games and see live pick percentages across all users - no login required, tracked by anonymous browser session ID.
MVP Voting
Anonymous one-vote-per-session MVP voting system. Results update in real time as votes come in throughout the tournament.
Snake Draft Board
An interactive snake draft board with a pick timer and real-time roster updates - used by team captains to select players before the tournament.
Online Registration
Player registration form with integrated Zelle payment flow and waiver tracking. Submission data feeds directly into the admin panel.
Admin Panel
A full admin interface for managing games, players, teams, and commissioner announcements - everything needed to run the tournament from one dashboard.
Player & Team of the Week
Weekly spotlights on the home page highlighting standout individual and team performances from the previous round.
Game Schedule
Full schedule view showing court assignments, tip-off times, round info, and live game status for every game in the tournament.
GroupMe Integration
Commissioner announcements sync with GroupMe so updates reach the whole community instantly through the messaging app they already use.
Mobile-First Design
The entire platform is built mobile-first with a bottom navigation bar - designed to be opened on a phone courtside during live games.

Project Successes & Failures

Building the core features of this site was a huge success, especially the real-time voting and the snake draft board. Getting a web application to update instantly across multiple devices without the user having to refresh the page was a major technical win for me. I was able to successfully implement a system that identifies a user's device to allow for "one-vote-only" polls and anonymous game predictions, which makes the site feel interactive and professional.

However, not everything went perfectly. During the live draft, I learned a tough lesson about how different devices talk to each other. I originally built a draft timer that ran on each person's phone, but because some phones were faster than others, the clocks got out of sync. Some phones showed they had ten seconds left when they actually only had two. So, I had rewrite the logic so that one central "master clock" in the database controlled everyone's screen.

ECE Skills Gained

Through this project, I have significantly expanded my software engineering skills. I learned how to build a modern, interactive frontend using React and how to manage complex, real-time data flows between a website and a cloud database. Understanding how to secure data and manage different levels of user access gave me a much stronger grasp of how professional web applications are architected. These software skills have given me a solid foundation for building the interfaces that modern hardware systems require.

As I continue this project, I will begin gaining hands-on hardware skills by connecting the website to a physical scoreboard. This transition into the "Internet of Things" (IoT) will teach me how to use microcontrollers to bridge the gap between digital code and physical electronics. I will be learning how to handle serial communication, sending specific strings of electrical pulses to turn on scoreboard lights, and how to manage power and signal integrity so that the scoreboard updates.

Next Steps

The most exciting next step is the completion of the physical scoreboard integration. I have an electronic scoreboard and am currently figuring out the best way to hook it up to my laptop. When an official updates the score on the website, the physical scoreboard on a table will update instantly. I'll be using an ESP32 microcontroller to act as the "brain" that catches the update from the internet and physically flips the digits on the scoreboard.

Beyond the hardware, I want to make the site smarter by adding automated features. Right now, I still have to do some manual work to set the playoff brackets. I plan to write an algorithm that looks at all the scores from the regular season and automatically seeds the teams into the bracket based on how well they played. This will make the tournament run even smoother and take the "human error" out of the equation, making the whole experience feel even more like a professional sports league.
