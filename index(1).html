GT Movies Store
January 2026

A full-stack movie marketplace built with Django 5. Users can browse a curated movie library, manage purchases, and submit reviews — while administrators have full control over platform data and moderation. I built this by intentionally typing every line of code by hand rather than copy-pasting, which gave me deep insight into how HTML, CSS, and Python interconnect within a Django project. Features include a custom review-reporting system I designed independently — not covered in the source textbook.

  • View application information via About page
  • Register an account and log in
  • Browse full movie catalog with images & titles
  • Search movies dynamically by title keyword
  • Add to cart, set quantities, and purchase
  • Clear shopping cart at any time
  • Create, edit, and delete personal reviews
  • View all reviews without logging in
  • View full order history with order numbers
  • Responsive UI for mobile, tablet, and desktop
  • Admin: manage users, movies, reviews, orders
  • Report inappropriate reviews (self-implemented feature)

CS 2340 · Object-Oriented Design
PandaPulse — LinkedIn-style Hiring Platform
Spring 2025 · 🏆 Top 5 / Both Sections · Final Exam Exempt

PandaPulse is a full-featured job search and hiring platform built in Django for CS 2340 (Objects and Design) at Georgia Tech. The platform serves three user roles — Job Seekers, Recruiters, and Administrators — and placed 3rd out of all submissions across both class sections, earning our team a final exam exemption. Beyond the 15 required user stories, we designed and implemented over a dozen original features that pushed the platform significantly beyond the baseline requirement.

  • Panda Paul — an integrated mascot chatbot that lets logged-in users navigate the site, update their profile, apply to jobs, and message people through natural conversation
  • "Pulses" Video Feed — short-form video posts (like Reels) where any user can promote themselves, their company, or share informative content; includes likes and deletion
  • Offer Comparison Tool — applicants can compare multiple job offers side-by-side with custom-weighted categories (salary, location, visa, benefits, etc.)
  • Interview Calendar — applicants schedule interviews from time slots set by the employer, fully integrated into the application workflow
  • One-Click Follow-Up — send a polished follow-up message after X days using pre-built templates
  • Skill Endorsements — employers can endorse applicant skills with one click after interviews
  • Response-Time SLA Badge — each employer shows a "responds in ~N days" badge on their listings
  • Resume Auto-Parser — uploaded resumes are parsed into profile fields with skill normalization for better job matching
  • Confetti on Apply — celebrates application submissions with confetti and fanfare
  • Application Viewed Indicator — applicants can see when an employer has opened their application
  • Private Notes — both applicants and employers can add private notes on applications
  • Rejection Feedback Templates — structured rejection messages with optional personalized notes
  • Applicant Activity Feed — real-time view of what's happening with each application
  • Company Culture Pages — company profiles featuring culture, perks, and office map gallery
  • Dark Mode — full site dark/light mode toggle persisted per user
  • Forgot Username / Password — email-based account recovery flow
  • Interactive Job Map — view all open roles pinned on a map; filter by commute radius
  • Kanban Recruiter Pipeline — drag-and-drop applicant management across hiring stages
  • Interview Feedback Workflow — structured internal feedback and decision rationale per candidate
  • Archived Rejections — employers can archive rejected candidates to revisit if new positions open
